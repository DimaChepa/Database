Для того, чтобы получить рабочий exe файл, необходимо:
1) Добавить в MSSQLSERVER базу данных, хранящуюся в файле OilStation.bak
2) В файле app.config сменить атрибут connectionStrings = "Data Source = --Имя вашего сервера, на котором будет храниться база данных"
3) В ide Visual Studio 2015 отладить проект
